## Jekyll 설치
- 공식 URL: <https://jekyllrb.com/>
- Linux 설치: <https://jekyllrb.com/docs/installation/>
- Windows 설치: <https://jekyllrb.com/docs/windows/>
- 한국어 번역: <https://jekyllrb-ko.github.io/>

## Local에서 Blog 실행
```bash
$ chcp 65001    # windows only
$ jekyll serve
```
주소: <http://localhost:4000>

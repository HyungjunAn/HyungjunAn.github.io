# 관사
- 국어에는 단복수 개념이 없어서(문맥에 의존하기에) 관사를 이해하기가 어려움

```
ex)
	I will buy a banana
	The banana in your hand.
	Bananas are fruits
	There is banana in this juice
```

## a, ~s
```
나는 바나나를 살 것이다.
	I will buy a banana
	I will buy bananas
```

## The
- 이미 합의된 또는 함께 인지하고 있는 대상을 지칭할 때 주로 쓰임
```
당신 손에 있는 바나나
	The banana in your hand.
```

## ~s
- 어떤 개체나 종 전체를 통틀어 말할 때 주로 쓰임
```
바나나는 과일이다
-> Bananas are fruits
```
	
## a, the, ~s를 안 쓰는 경우
- 갯수를 특정지을 수 없을 때

```
이 주스에는 바나나가 들어 있다.
	There is banana in this juice
```
